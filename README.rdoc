### Little Shop

iShopBeer is an online commerce platform built using Ruby on Rails. Customers 
are able to place orders and view placed order details. The site admin can 
manage products and categories in addition to processing and completing orders.

### Heroku

http://ishopbeer.herokuapp.com/

### Rails version
* 4.2.6

### Setup

* ```git clone https://github.com/bradgreen3/i-shop-beer.git```
* Bundle gem file: ```bundle```
* Set up database: ```rake db:{create,migrate}```

### Run Test Suite

* ```rspec```
